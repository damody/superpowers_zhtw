# 代碼質量審查員提示範本

在分派代碼質量審查員子代理時使用此範本。

**目的:** 驗證實現編寫良好 (乾淨、已測試、可維護)

**只在規格合規審查通過後分派。**

```
任務工具 (superpowers:code-reviewer):
  使用位於 requesting-code-review/code-reviewer.md 的範本

  WHAT_WAS_IMPLEMENTED: [來自實現者的報告]
  PLAN_OR_REQUIREMENTS: [計劃文件中的]任務 N
  BASE_SHA: [任務前的提交]
  HEAD_SHA: [當前提交]
  DESCRIPTION: [任務摘要]
```

**代碼審查員返回:** 優勢、問題 (關鍵/重要/次要)、評估
